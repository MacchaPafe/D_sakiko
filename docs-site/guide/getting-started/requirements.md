# 运行要求

在下载软件前，请先确认你的电脑系统和硬件是否适合运行数字小祥。

因为数字小祥需要同时运行 AI 对话、语音识别、声音合成以及 Live2D 等多个模块，所以她比普通的聊天软件需要更多的内存和计算资源。如果你的电脑没有达到推荐条件，程序可能还是能打开，但是生成声音会变得特别慢，或者部分功能干脆没法用。

## Windows 系统

我们建议你使用：

- 64 位的 Windows 操作系统；
- NVIDIA（英伟达）独立显卡；
- 预留大约 10 GB 的磁盘空间（用于解压软件包和保存日后的聊天数据）；
- 稳定的网络连接。

为了获得最好的体验，建议使用显存（VRAM）达到 6 GB 或更高的 NVIDIA 显卡。

### 如果我没有 NVIDIA 显卡能玩吗？

可以的。程序会尝试用你的 CPU 来生成声音，但速度通常会明显变慢。

要注意的是，任何显卡都可以正常显示 Live2D 的画面，但在目前的 Windows 软件包里，只有 NVIDIA 显卡能利用 CUDA 技术来给声音合成加速。

::: tip 怎么看自己的电脑配置？
在 Windows 里打开“设置 → 系统 → 系统信息”，可以查看你的处理器（CPU）和内存；
按下 `Ctrl+Shift+Esc` 打开“任务管理器”，切到“性能”标签页，就能查看你的显卡型号和显存大小。
:::

## macOS 系统

目前 macOS 软件包的正式支持条件是：

- **Apple M 系列芯片**（例如 M1、M2、M3 或 M4）；
- **macOS 14 Sonoma 或更高版本的操作系统**；
- 能够正常访问我们的软件下载网盘；
- 至少预留 8 GB 的磁盘空间。

如果你还在使用老款的 Intel 芯片的 Mac，目前是不在我们的正式支持范围内的。

### 怎么看我的 Mac 是什么芯片？

1. 点击屏幕左上角的苹果图标 ；
2. 选择“关于本机”；
3. 查看弹窗里的“芯片”和“macOS”这两行。

如果“芯片”那一栏写着 Intel，就不符合目前 macOS 软件包的运行要求了。

## 网络要求

数字小祥的以下功能是需要联网的：

- macOS 第一次安装环境依赖时；
- 使用网上的 AI 模型和你聊天时；
- 角色使用网络搜索、查天气等工具时；
- 你在软件里下载新的 Live2D 衣服或者其他资源时；
- 检查和下载软件更新时。

对于 macOS 来说，第一次安装时大概需要下载 1-2 GB 的东西。如果你网速不快，可能需要多等一会儿。

Windows 的软件包虽然已经把运行环境都打包好了（不需要额外下载依赖），但在日常聊天和用网络工具时，依然是需要全程联网的。

## 磁盘空间与安装位置

请务必把软件放在一个空间够大，且你自己有读写权限的文件夹里。因为随着你每天和角色聊天，软件还会不断生成并保存以下文件：

- 你们的聊天记录；
- 角色生成的声音音频；
- 运行日志；
- 你下载的 Live2D 模型和衣服；
- 升级时的临时文件。

### Windows 下的路径限制

在 Windows 上，**请千万不要把软件放在带有中文或特殊字符的路径下**。

**推荐放在这里：**
```text
C:\Users\xxxx\Documents\D_sakiko
D:\Apps\D_sakiko
```

**绝对不要放在这里：**
```text
C:\Users\张三\Desktop\数字小祥
D:\我的游戏\数字小祥
```

如果你的 Windows 登录用户名本来就是中文（比如叫“张三”），那么你的桌面和“下载”文件夹路径里就会带有中文。遇到这种情况，请直接在 D 盘或 C 盘的根目录下新建一个纯英文的文件夹来放它。

## 实在不确定能不能跑怎么办？

如果你的电脑大致符合上面的要求，但你还是担心带不动，可以先下载安装，试着发一条文字消息看看。

如果打字聊天很流畅，但生成语音卡了半天，你可以把“语音合成”关掉，当个纯文字版的桌宠玩；或者在“语音合成模型配置”里，尝试调低推理精度和采样步数。
