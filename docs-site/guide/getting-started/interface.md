# 界面与基本操作

这篇文档会带你熟悉主聊天窗口中最常用的功能区。不同操作系统的电脑、不同的窗口大小和主题颜色下，按钮的具体位置可能会稍微有点变化，但它们的功能和图标是基本一致的。

## 主聊天区域

窗口正中间是显示你和角色对话记录的地方。当角色开始回复时，文字会一个字一个字地蹦出来；等整段话生成完，程序就会开始播放角色的声音。

### 对消息的操作（点击和右键）

如果你想再听一遍角色说过的话，直接**左键点击**那条角色的消息就可以回放声音了。

如果你对某条消息**点击右键**，根据消息是谁发的以及位置的不同，你会看到以下菜单：

- **编辑消息**：修改这句话的内容；
- **编辑并重发**：修改你发的话并让角色重新回答；
- **重新生成本轮回复**：对角色的回答不满意？让她换个说法重答一次；
- **回溯到此处**：把聊天进度退回到这条消息的状态（后面的记录会被删除）；
- **删除单条消息**；
- **删除整轮对话**：把你的提问和角色的回答一起删掉；
- **重新生成角色语音**：文字没问题但声音卡了或变调了，可以点这个重新合成一遍声音。

请注意，删除和回溯操作是会实实在在删掉后续聊天记录的，点确定前请看清弹出的提示。

## 输入和发送

在窗口最底部的输入框里打字：

- 写完后按键盘上的 `Enter` 发送；
- 如果想换行，可以使用 `Shift + Enter` 或 `Ctrl + Enter` 等组合键；
- 也可以直接点击输入框右下角的小箭头按钮发送。

当角色正在费力思考和回复时，原来的发送按钮会变成一个**停止按钮**，点它就能打断角色的发言。

另外，你还可以直接把电脑里的文件拖进输入框里。不过，AI 到底能不能读懂这个文件，还要看你选的模型聪不聪明，以及软件当前支不支持读取这类文件。目前由于小祥的限制，模型是无法读取并理解你给出的图片的。

## 语音输入（麦克风）

输入框旁边的麦克风按钮用起来和微信语音差不多，是“按住说话、松开识别”的模式：

1. 鼠标按住麦克风按钮不放；
2. 看到窗口标题变成了“正在录音...”后，开始说话；
3. 说完后松开鼠标；
4. 等一会儿，程序会把你刚刚说的话变成文字填到输入框里；
5. 你可以检查和修改一下这些文字，确认没问题后再发给角色。

如果你录的时间太短，程序会忽略这次录音。另外，在刚打开软件、语音识别模型还没准备好时，这个按钮是灰色的，不能点。

## 推理模式（深度思考）

输入框旁边还能找到一个控制“推理模式”的按钮。这决定了角色在回答时会不会进行“深度思考”：

- **自动**：随缘，看 AI 平台决定是否开启推理；
- **思考**：强迫它开启推理模式，想清楚再答；
- **快速**：关闭推理模式，赶紧回话；
- **推理强度**：可以选择思考的深浅程度。

这个思考设置是跟着每个不同的对话单独保存的。不过要注意，不是所有 AI 模型都支持这些高级功能，如果你选的模型不支持，设置了也没用。具体请看 [AI 模型设置](../llm.md)。

## 记忆力指示（上下文用量）

输入框附近有个小圆环，它显示了当前的这段聊天大概占用了 AI 多少“记忆力”（也就是模型上下文 Token）。你可以点一下这个圆环看具体的数字。

你们聊得越久、设定的背景故事越长，圆环填满得就越快。如果程序不知道你选的这个新 AI 的记忆力上限是多少，圆环就只会显示已经用了多少，不会显示上限。

## 对话列表（历史聊天）

点击界面顶部中间的“对话列表”按钮，可以把右侧的聊天历史面板展开或收起来。

在这个侧边栏里，你可以：

- **新建对话**：找角色开一个全新的聊天话题；
- **删除当前对话**；
- **在不同的聊天记录之间切换**；
- 鼠标按住拖动来**重新排序**聊天记录；
- 右键点击某条聊天来**重命名**或**删除**它；
- 切换显示模式（是全部平铺开，还是按角色折叠起来）。

每次新建对话时，你需要选好：
- 找哪个角色聊；
- 给这个话题起个标题；
- 决定要不要带上你设定好的“用户身份”。

数字小祥很聪明，每段独立的对话都会记住自己当初用的角色、你的身份以及思考设置。

## 顶部状态与快捷按钮

主聊天窗口最上面有一排重要的信息和按钮：

- **状态提示**：告诉你当前是“就绪”可以聊天了，还是正在生成回复，又或者是出了什么错误；
- **保存聊天记录**：一般软件会自动保存，但你不放心时可以点这个手动存一下；
- **对话列表**：用来开关右侧的历史面板；
- **设置（齿轮图标）**：调整当前聊天和角色外观的地方；
- **更多功能（四个方块）**：通往高级设置和其他独立小程序的入口。

## 当前聊天设置（齿轮按钮）

点击齿轮图标，你可以调整很多马上就能看到效果的设置：

- 规定角色用中文还是日文回答你；
- 开关角色的声音合成；
- 一键清空当前的聊天记录；
- 决定 Live2D 画面上要不要显示字幕；
- 给角色换个 Live2D 的背景图片；
- 调节 Live2D 的流畅度（帧率）；
- 换一下聊天窗口的主题颜色；
- 给当前角色换个声音（参考音频）；
- 给当前角色换一套衣服（Live2D 模型）；
- 对于祥子这个角色，还能开启她的特殊状态和面具。

有些按钮如果变灰不能点，说明你可能还没下载相关的衣服或声音文件。

## 语速和停顿调节

在输入框上面还有两个很实用的滑块：

- **语速调节**：嫌角色说话太慢或太快？调它。
- **句间停顿时间**：角色说长篇大论时，两句话之间喘气的间隔时间。

这两个滑块主要是方便你一边听一边微调的。不建议拉到最极端的值，否则声音听起来会很像机器人。

## 更多功能（四个方块按钮）

点击“更多功能”图标，会弹出一个菜单，带你去这些地方：

- 去编辑角色的动作表情；
- 去修改你的人设和角色的设定；
- **启动参数配置**（很重要，换大模型和调全局设置都在这里）；
- 进入小剧场模式；
- 打开 Live2D 模型下载器；
- 检查有没有新版本可以更新；
- 安全地**退出程序**。

如果你想换更好的大模型、调声音推理参数或者换字体主题，记得去“启动参数配置”（也叫设置中心），详情可以看 [设置中心与个性化](../settings.md)。
