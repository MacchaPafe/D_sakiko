# Windows 安装

我们的 Windows 软件包是“开箱即用”的，你只需要下载并解压就可以开始使用了。

::: tip 开始前
请先看一眼前面的 [运行要求](./requirements.md)，特别是确认一下你准备放软件的路径里**绝对不能有中文**。
:::

## 1. 下载 Windows 软件包

你可以从下面这几个网盘里挑一个下载：

- **百度网盘**：https://pan.baidu.com/s/1kLJT0mZ_gGQ2yRUrMJU_Tw?pwd=7vsd 提取码: 7vsd
- **天翼云盘（不限速）**：https://cloud.189.cn/t/b2EjU36Nj67b （访问码：0fe8）
- **Google Drive**：https://drive.google.com/drive/folders/1jWgKRWXvpRIgyiYpkVgk7vXGMFL-kiyz?usp=sharing

## 2. 完整解压

下载完之后，右键点击压缩包，选择“全部解压”，或者用你电脑里的解压软件（比如 Bandizip、WinRAR），把整个软件包解压到一个新的文件夹里。

如果你是从百度网盘下载的，你通常会得到三个后缀名为 `.001`、`.002`、`.003` 的分卷压缩包。
这个时候，**请确保这三个文件都放在同一个文件夹里**，然后只对 `.001` 那个文件右键选择“解压到当前文件夹”。解压软件很聪明，它会自动帮你把后面的 `.002` 和 `.003` 合并解压出来。

请再次确认，你解压后的软件文件夹路径里没有中文或特殊字符，例如这样是没问题的：

```text
C:\Users\xxxx\Documents\D_sakiko
D:\Apps\D_sakiko
```

::: warning 绝对不要在压缩包里直接双击运行
数字小祥在运行时需要同时读取成百上千个模型、字体和配置文件。如果你偷懒，直接在压缩软件的预览窗口里双击运行，或者只把里面的 `run.bat` 拖到桌面上，软件绝对会报错崩溃的。
:::

解压完成后，请不要随意移动或删除文件夹内部的东西。

## 3. 启动程序

打开你解压好的文件夹，双击这个文件：

```text
run.bat
```

程序会首先弹出一个黑色的命令行（终端）窗口，然后开始加载各种环境和模型。在这个过程中，**不要把黑窗口关掉**。

## 4. 等待窗口出现

第一次启动时间较长，大概需要等上 1 到 2 分钟。你会按顺序看到：

- 黑色的命令行窗口；
- 主聊天窗口；
- 角色的 Live2D 画面窗口。

当主聊天窗口上方显示“就绪”后，你就可以参考 [第一次启动](./first-run.md) 的说明，给角色发一条测试消息了。

## 5. 以后想玩的时候怎么启动？

以后每次想找数字小祥聊天，依然是去那个文件夹里双击 `run.bat`。

如果你觉得每次都要点进文件夹太麻烦，可以右键点击 `run.bat`，选择“发送到 → 桌面快捷方式”。（切记是创建快捷方式，而不是把原文件剪切到桌面）。

## 常见的安装小麻烦

### 双击 `run.bat` 后一点反应都没有

先排查这三点：

1. 你的压缩包是不是完整解压出来了？
2. 放软件的文件夹路径里是不是不小心带了中文？
3. 你电脑里的杀毒软件（比如 360、火绒）是不是偷偷把这个批处理文件给拦截了？

如果黑窗口只是一闪而过，你可以试着多点几次，或者眼睛睁大点，看看它闪过的那一瞬间报了什么错。

### 那个黑窗口好烦，我能关掉吗？

**不能。** 黑窗口里跑着的是整个软件的核心大脑。如果你把它叉掉了，主聊天窗口也会跟着死掉退出的。
如果你不想聊了，正确的退出姿势是：在主窗口点击“更多功能 → 退出程序”。

### 我的电脑没有 NVIDIA（英伟达）显卡怎么办？

程序会尝试用你的 CPU 来生成声音，只是速度可能会比较慢。你也可以先在输入框输入 /v 并回车，关掉“语音合成”，专门使用文字聊天。
