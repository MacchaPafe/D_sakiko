# 第一次启动

## 如何启动程序

- **Windows 系统**：双击软件文件夹里的 `run.bat`；
- **macOS 系统**：双击软件文件夹里的 `运行主程序.command`。

双击之后，屏幕上会出现一个黑色底的命令行（终端）窗口。请注意，这个黑窗口是程序运转的核心部分，**在软件运行期间千万不要关闭它**。

稍等片刻，通常会自动弹出两个新窗口：
- 主聊天窗口；
- 角色模型（Live2D）窗口。

因为第一次启动时程序需要解压和加载多个模型，所以速度会比以后慢不少。请耐心等待，直到主聊天窗口上方的状态提示变成“就绪”。

## 了解启动过程

哪怕窗口已经弹出来了，有些功能可能还在后台继续做准备：

- 语音识别模型没加载完之前，聊天框旁边的麦克风按钮会是灰色的（不可用）；
- 第一次让角色开口说话时，因为需要实时加载声音模型，所以可能会卡顿一会儿；
- 角色的 Live2D 窗口可能会比聊天窗口晚一点才显示出来。

只要窗口没有卡死（能拖动），且黑色终端窗口里没有刷出明显的错误提示，就请多给它一点时间。第一次启动大概需要 30 秒到 1 分钟左右。

## 发送第一条消息

1. 点击主聊天窗口底部的输入框；
2. 输入一句简短的打招呼，比如“你好”；
3. 按下键盘上的 `Enter`（回车键），或者点击右下角的发送按钮；
4. 等待角色的回复和语音。

如果你想在输入框里换行，可以按 `Shift + Enter` 或者 `Ctrl + Enter`。

数字小祥默认提供了一条免费的 AI 模型测试线路，所以你在第一次测试时不需要急着去填自己的 API Key。

## 怎样才算成功运行了？

你可以对照以下几点来检查：

| 检查项目 | 正常的表现是怎样的 |
| --- | --- |
| 主窗口 | 能够正常打字并发送出去。 |
| AI 回复 | 聊天界面里出现了角色回复的文字。 |
| Live2D | 角色窗口正常弹出来，能看到角色。 |
| 角色语音 | 角色回复文字后，你能听到她说话的声音。 |
| 麦克风 | 稍等一会儿后，麦克风按钮变亮，可以按住说话。 |

注意：文字回复、语音和画面是相对独立的。举个例子，如果角色回复了文字但没有声音，通常说明 AI 模型没问题，而是你的声音合成没设置好或者正在加载。

## 第一次测试失败了怎么办？

### 一直等不到回复

请去那个黑色的终端窗口或者主聊天窗口的提示里看看，有没有提到网络不通、API Key 错误、余额不足之类的信息。如果默认的免费线路出问题了或者提示没额度，你可能需要去设置里配置自己的 API Key。

### 有文字回复，但是没声音

首先去设置里确认一下你没有误关掉“语音合成”的开关，然后稍微耐心等一下，第一次合成声音时模型加载得比较慢。

### 没看到角色的画面（Live2D 没有显示）

确认一下你是不是不小心把 Live2D 的窗口关了。另外检查一下终端（黑窗口）有没有显示错误，如果有错误的话，可以找我们[反馈](../feedback.md)。记得带上日志！

## 找找设置在哪里

在主聊天窗口的右上方，你会看到两个不同的按钮：

- **设置（齿轮图标）**：主要用来调整当前的聊天，比如让角色说中文还是日语、开关声音、换背景、换参考音频等；
- **更多功能（四个方块图标）**：这里包含“启动参数配置”（即设置中心）、编辑用户人设、小剧场、下载衣服等功能。

如果你想换 AI 模型、调整声音推理参数或是更改界面字体和主题颜色，请点击“更多功能 → 启动参数配置”。详细说明可以看 [设置中心与个性化](../settings.md)。

## 怎么正常退出软件？

我们建议的退出方式是：在主窗口点击 **更多功能 → 退出程序**。

这样能让程序优雅地保存你的聊天记录和设置，并安全地清理后台进程。请**不要**直接去点那个黑色终端窗口的叉号来强行关闭。
