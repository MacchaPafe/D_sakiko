# 设置中心与个性化

设置中心是我们为你提供的控制台，你可以在这里管理数字小祥的 AI 模型、语音合成、角色的登场顺序以及界面外观，还可以顺便设置你自己的人设信息。

::: tip 这篇文档讲什么？
这篇文档主要带你了解设置中心的页面结构、每一项设置什么时候生效，以及如何调整角色顺序、更换字体和主题颜色等个性化功能。

至于 AI 模型、语音合成和用户人设的具体细节，由于内容较多，我们有专门的说明文档，这里只简单介绍它们的入口。
:::

## 如何打开设置中心

如果主程序已经在运行，你可以点击右上角的“设置”图标，然后选择“启动参数配置”。

如果你还没有运行主程序，也可以直接从软件文件夹中打开“启动参数配置”。不同电脑系统下的文件名可能不太一样：

- Windows 系统：双击运行 `启动参数配置.bat`
- macOS 系统：双击运行 `启动参数配置.command`

打开之后，你会看到窗口左侧有两个主要菜单：

- **角色与用户人设**：用来查看系统自带的角色，以及创建和修改你自己的“用户人设”；
- **设置**：用来配置 AI 模型、语音合成以及其他个性化选项。

## 设置页面一览

在进入“设置”后，页面上方有三个标签页供你选择：

| 标签页 | 可以设置什么 | 去哪看详细说明 |
| --- | --- | --- |
| 大模型 API 配置 | 模型平台、模型名称、API Key、温度、核采样等 | 请看 [AI 模型设置](./llm.md#configuration) |
| 语音合成模型配置 | 推理精度、采样步数、GPU 加速、模型加载数量等 | 请看 [语音合成](./tts.md) |
| 个性化 | 角色的登场顺序、自定义字体、界面主题和强调颜色 | 本文下方有详细介绍 |

在这个页面里，你可以：

- 查看系统中的角色；
- 设置角色以后该怎么称呼你；
- 填写你自己的性格、背景故事和说话习惯；
- 如果你想，还可以选择“扮演已有角色”，用系统角色的身份去聊天。

请注意，“对话身份”里的名称和背景描述会直接影响 AI 怎么看待你。而你设置的头像只用于界面展示，它不能代替文字描述，也不会让 AI 自动“看图认人”。

更完整的说明，请参考“对话身份”文档。

### 大模型 API 配置

这里用来挑选和配置负责理解消息并回话的 AI 模型。你可以：

- 试用我们提供的免费模型；
- 选择 DeepSeek、Gemini、OpenAI 等内置平台；
- 接入 ModelScope（魔搭社区）；
- 使用自定义 API 接入兼容 OpenAI 格式的其他服务；
- 调整模型温度等高级参数；
- 开关“显示无法解析的模型原文”。

不同模型在角色扮演、工具调用和回复速度等方面会有很大不同，具体的选择建议请参阅 [AI 模型设置](./llm.md#configuration)。

### 语音合成模型配置

这里控制声音模型在生成角色语音时，如何利用你的电脑硬件。你可以设置：

- 退出程序时是否删除历史生成的音频；
- 使用 FP16 还是 FP32 推理精度；
- 声音合成的采样步数；
- 是否开启显卡加速（NVIDIA CUDA 或 Apple MPS）；
- 最多同时加载几个角色的声音模型。

这些设置会直接影响语音生成的快慢、音质以及对电脑内存和显存的占用。对于不同的电脑硬件该怎么选，请参阅 [语音合成](./tts.md) 文档。

## 保存设置

当你修改完设置后，别忘了点击窗口底部的“保存配置”按钮。保存成功后，底部会弹出以下两种提示之一：

- **保存成功！已实时更新配置**：说明主程序已经立刻收到通知并启用了新设置。
- **保存成功！配置在下次启动时应用**：说明这些设置已经存入文件，但你需要重新启动主程序才能看到变化。

::: warning 注意
有些设置涉及到语音模型和硬件设备的重新初始化，即便提示了“实时更新”，也可能需要重启一次程序才保险。具体请参考旁边每一项的文字说明。
:::

## 什么时候会生效？

为了方便你查阅，我们整理了常见设置的生效时机：

| 设置项 | 什么时候生效 |
| --- | --- |
| AI 模型服务商、模型名称和 API Key | 通常可以实时更新 |
| 模型温度、Top-p、显示无法解析的原文 | 通常可以实时更新 |
| 删除历史音频 | 保存后，在下一次退出程序时生效 |
| FP16 / FP32 推理精度 | 重启主程序后生效 |
| 语音合成采样步数 | 重启主程序后生效 |
| CUDA / MPS 硬件加速 | 重启主程序后生效 |
| 同时加载语音模型数量上限 | 重启主程序后生效 |
| 角色登场顺序 | 建议重启主程序后确认 |
| 自定义字体 | 重启主程序后生效 |
| 浅色/深色主题和界面强调色 | 保存后立刻显示效果 |

如果修改后发现没变化，你可以先“保存配置”，然后把主程序关掉重新打开试试。

### 调整角色登场顺序

这里的顺序决定了主程序中角色列表的排列先后。

1. 展开“角色登场顺序”面板；
2. 按住某个角色，拖动到你想要的位置；
3. 调整完后，点击“保存配置”；
4. 重新启动主程序，去看看角色顺序是不是变了。

如果你后来添加了自定义角色，也可以随时来这里给他们重新排座次。请放心，这里只改变列表排序，不会弄乱角色的设定或者声音文件。

### 使用自定义字体

点击“自定义字体”旁边的“选择”按钮，你就可以从电脑里挑选喜欢的字体了。

目前支持以下字体格式：

- `.ttf`
- `.otf`
- `.ttc`

选好之后，设置程序会把字体文件复制到数字小祥专门的 `font` 文件夹中，并自动加上时间标记防止冲突。主程序在启动时会寻找最新导入的这个字体，因此**换完字体后必须要重启主程序**。

另外，程序会自动尝试帮你清理以前导入的旧字体。如果那个旧字体正好被其他软件占用导致删不掉，不用担心，这不会影响你新字体的正常使用。

::: warning 字体兼容性提醒
有些比较冷门的字体可能没有做全中日文字符。如果在软件里看到文字变成了空白、方框，或者夹杂着别的字体，说明这款字体的字库不全。建议你换一款包含完整中日文字形的常见字体。
:::

### 切换应用主题

“应用主题”控制着设置界面是深色还是浅色，有三个选项：

- **浅色**：强制使用白底的浅色界面；
- **深色**：强制使用黑底的深色界面；
- **自动**：跟随你电脑系统当前的颜色设置。

请注意，这里调的只是“设置中心”以及其他相关功能界面的外观，**不会**影响聊天界面的 Live2D 背景或角色的衣服颜色。

### 选择主题颜色

主题颜色指的是界面里的按钮、开关和选中项的高亮颜色。我们为你准备了多种以角色名字命名的主题色，你也可以随心所欲地调配自己的专属颜色。

需要说明的是，这里调的颜色**仅对设置界面有效**。在主程序的聊天窗口里，整体的配色会根据你当前聊天的角色自动变化，不受这里的控制。

展开“主题颜色”后，你可以：

- 挑选一个预设主题；
- 修改主题的名字；
- 重新编辑主题颜色；
- 点击“+”添加新主题；
- 删除看腻了的旧主题。

当你选中某个主题后，设置界面的按钮颜色会立刻跟着改变，不用重启。

::: tip 界面外观设置的区别
- **应用主题**：控制界面底色是亮还是暗。
- **主题颜色**：控制界面按钮等小元素的点缀颜色。
- **聊天背景和服装**：这些属于 Live2D 功能，你需要去另外的地方换，不受上面这两个设置的影响。
:::

## 配置文件与隐私安全

我们的大部分设置都保存在程序文件夹下的 `d_sakiko_config.json` 文件里，其中也包含了你填写的 API Key 等敏感信息。因此：

- **千万不要**把这个文件直接发到群聊、公开 Issue 或其他任何人都能看到的地方；
- 如果你要截图反馈问题，请务必给你的 API Key 打上马赛克；
- 换电脑需要迁移配置时，请用安全私密的方式传输这个文件；
- 万一 API Key 泄露，请立刻去服务商的官网作废它，并重新生成一个新的。

另外，我们强烈建议你**不要用记事本去强行修改这个文件**。请始终使用“设置中心”来操作，这样能避免拼写错误或弄坏文件格式导致软件打不开。
