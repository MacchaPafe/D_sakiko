# 开始使用

欢迎使用数字小祥！本章节会带你一步步完成运行条件检查、软件安装、第一次启动以及了解界面的基本操作。

## 推荐阅读顺序

为了让你能顺利开始聊天，建议按照以下顺序阅读：

1. 先看看[运行要求](./getting-started/requirements.md)，确认你的电脑是否符合条件；
2. 按照 [Windows 安装](./getting-started/windows.md) 或 [macOS 安装](./getting-started/macos.md) 的说明完成安装；
3. 阅读[第一次启动](./getting-started/first-run.md)，尝试向角色发送你的第一条消息；
4. 阅读[界面与基本操作](./getting-started/interface.md)，熟悉聊天窗口和设置在哪里。

## 选择你的操作系统

数字小祥同时支持 Windows 和 macOS。请根据你使用的电脑系统，选择对应的安装文档：

### Windows

Windows 软件包是“开箱即用”的，里面已经包含了运行所需的所有环境。下载并完整解压后，就可以直接运行了。

[前往 Windows 安装教程 →](./getting-started/windows.md)

### macOS

macOS 软件包在第一次使用前，需要先运行一个自动安装脚本，用来配置必要的运行环境。目前我们正式支持 Apple M 系列芯片的 Mac。

[前往 macOS 安装教程 →](./getting-started/macos.md)

## 安装完成后可以做什么？

完成第一次启动并成功收到角色回复后，你还可以探索以下功能：

- 配置自己的 AI 模型（API Key）；
- 创建并完善你自己的“用户人设”；
- 添加自定义的角色和声音模型；
- 为角色下载和更换 Live2D 服装；
- 调整字体、主题配色以及声音合成的细节设置；
- 尝试使用“小剧场模式”。

不用急着在第一次启动时就把所有设置都搞懂，建议先确认默认角色能正常和你聊天，之后再去慢慢尝试这些高级功能。

## 应该下载软件包还是源代码？

如果你只是想把数字小祥当做一个桌面宠物和聊天软件来使用，请直接下载已经打包好的 **Windows 或 macOS 软件包**。普通用户完全不需要自己去安装 Python、Git 等开发工具。

如果你希望修改代码、参与开发，或者由于某些原因需要从源代码运行程序，请参阅我们的 [开发环境设置](../dev/setup.md) 文档。

| 你的目标 | 我们推荐的方式 |
| --- | --- |
| 只是想聊天，体验现有功能 | 下载对应系统的软件包 |
| 想修改角色、模型或界面设置 | 仍然下载软件包，通过程序里自带的设置中心来完成 |
| 想修改程序源代码或参与开发 | 按照开发环境设置指南，从源代码运行 |
