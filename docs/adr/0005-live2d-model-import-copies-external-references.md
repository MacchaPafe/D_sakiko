# Live2D 模型导入复制外部引用到 extra_model

Live2D 模型导入只把项目外部的 `.model.json` 或 `.model3.json` 导入到当前角色的 `extra_model/{唯一模型名}`，不覆盖默认 `live2D_model`，也拒绝导入当前 `live2d_related` 目录内已有模型，避免循环导入。导入器通过解析模型 JSON 的引用清单复制 V2/V3 所需文件，并把所有引用重写为目标目录内相对路径；普通相对路径尽量保留结构，绝对路径或带 `..` 的外部引用收敛到安全子目录。

导入器只负责把外部模型复制成项目内自包含模型，后续项目规范化仍交给 normalizer。导入成功后只刷新模型列表并提示用户选择，不自动切换模型；导入失败时删除本次新建的目标目录，不支持覆盖已有模型目录。
