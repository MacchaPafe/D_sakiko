# Qt 相关 ui 组件命名约定

为了区分不同的 Qt 组件，尽可能复用一些类似的组件，我们在组件存放位置上做一些约定：

- custom_widgets 文件夹：存放通用的自定义 Qt 组件。这些组件通常是通用的小组件，可以用于各种业务、各种 UI 界面的构建，比如“可以多选的 QComboBox”、“背景默认为透明的 QScrollArea”等等。
- components 文件夹：存放用于特定业务功能的 Qt 组件。这些组件通常是为特定功能设计的，虽然也可以出现在多处，但只能用于单个功能。例如，“Codex 风格的命令选择栏（和主界面样式、注册命令强绑定）“，”角色对话选择侧栏（只能用于对话切换）“
